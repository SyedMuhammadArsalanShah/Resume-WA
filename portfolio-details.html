Al-Mushaf | Quran

Project information

  • Category: Mobile App
  • Tools: Flutter | Quran API
  • Project date: 07 May, 2023

Project Description

  1. Quran Text Display: Display the Quranic verses in an elegant and readable format with options for various translations and Arabic scripts.
  2. Audio Quran Integration: Utilize a suitable package (e.g., audioplayers) to seamlessly integrate the audio recitation of the Quran with options for different reciters.
  3. Audio Controls Placement: Position audio playback controls (play, pause, volume) conveniently on the screen for easy access while listening to Quranic recitations.
  4. Gesture Controls: Enable swipe gestures for easy page-turning or verse-scrolling within the app.
  5. Progress Indicator: Display a progress bar or indicator to show users their current location within the Quranic text and audio.
